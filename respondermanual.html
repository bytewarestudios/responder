Responder allows you to set any number of column combinations within a row as long as they all add up to 100%.� Available classes for column percentages are listed below.

Class Name

Description

Makes a column 12.5% of a rows width or 1/8th of a row width

column33

Makes a column 1/3rd of a rows width or 33.33% repeating
